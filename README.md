## Work In Progress

This is a vibecoded Next.js Youtube Frontend built for educational and demonstrational purposes, it uses Next server to store API key and acts as a middleware between frontend and Youtube Backend

Deployment: https://project-t6oqu.vercel.app/

## Stack

- base:
  - Typescript
- framework:
  - Next.js
- state management:
  - Redux Toolkit / React-Redux
- API requests
  - RTK Query: internal API requests
  - googleapis: API requests to Youtube backend
- UI
  - TailwindCSS
  - shadcn/ui
  - next-themes
- Event handling
  - Mitt: event bus
